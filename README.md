 student-registration-system
it a is student registration system created with html java and Css

A lightweight, responsive web application for managing student records. This project allows users to register student details, view them in an organized table, and perform Edit/Delete operations with data persistence.
 Features
Functionality: Create, Read, Update, and Delete student records.
Data Persistence:uses localStorage to ensure data remains available even after page refreshes.
- Form Validation
  - Names accept only characters.
  - IDs and Contact Numbers accept only digits.
  - Contact Numbers must be at least 10 digits.
  - Email format is strictly validated.
  #Responsive Design:Fully optimized for Mobile (≤640px), Tablet (641px–1024px), and Desktop (≥1025px).
  Dynamic UI: Includes a dynamic vertical scrollbar that appears automatically as the record list grows.

Technologies Used
- HTML5: Semantic structure for better accessibility.
- CSS3: Custom styling with Flexbox and Media Queries for responsiveness.
- JavaScript : DOM manipulation, event handling, and local storage management.

# Project Structure
- index.html - The main entry point containing the form and display section.
- style.css - Contains all layout and design styling.
- script.js - Contains the business logic, validation, and storage functions.
- README.md - Project documentation.

# How to Use
1. Clone the repository or download the zip file.
2. Open "index.html" in any modern web browser.
3. Fill in the student details in the registration form.
4. Click "Add Student" to save the record to the table.
5. Use the "Edit" button to modify an existing entry or "Delete" to remove it.
